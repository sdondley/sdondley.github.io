## My Notes
* string comparison
  * `"foo" ==? "FOO"`
    *  "case-**insensitive** no matter what the user has set"
  *  `"foo" ==# "FOO"`
    *  "case-**sensitive** no matter what the user has set"
  * test if not equal
    * `!=?`
    * `!=#`
